## Installation

Before installing, [download and install Node.js](https://nodejs.org/en/download/).

## Quick Start

The quickest way to setup Sneaker app is as shown below:

```
git clone git@github.com:AJAkimana/ivugurura.git
cd ivugurura
```

Install dependencies:

```bash
$ npm install
```

Start the server:

```bash
$ npm run start
```
